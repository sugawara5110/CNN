・PPMLoader

  PPMファイルの読み込みを行います
  
  引数にpass, ファイル1個の出力サイズ, 
  
  画像モードを入力することで
  
  指定のpassから全PPMファイルを読み込み,
  
  指定出力サイズにリサイズし1次配列で出力します。
  
  取得ファイル数も出力されます。
  
  画像モードでNormalの場合 1ピクセルに付き3byte(フルカラー)
  
  画像モードGrayScaleの場合 1ピクセルに付き1byte(モノクロ)
  
  で出力します
